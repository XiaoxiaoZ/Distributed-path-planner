# Distributed-path-planner
Distributed Path Planner based on elixir

rust is used as erlang nif

## Installation

Adding `dpp` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:dpp, "~> 0.1.1"}
  ]
end
```
